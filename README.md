# 🖥️ Детекция ноутбуков на партах

Веб-приложение для автоматической детекции ноутбуков на изображениях с использованием модели YOLOv8.

---

## 📋 Описание

Приложение позволяет загрузить изображение учебной аудитории и автоматически обнаружить все ноутбуки на нём. Результат отображается с ограничивающими рамками вокруг каждого найденного объекта и статистикой (количество ноутбуков). Все запросы сохраняются в истории.

---

## 🛠️ Технологии

| Компонент | Технология |
|:---|:---|
| Язык программирования | Python |
| Веб-фреймворк | Flask |
| Нейросетевая модель | YOLOv8 |
| Обработка изображений | OpenCV |
| База данных | SQLite |
| Интерфейс | HTML5, CSS3, Bootstrap |

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/leandreev1234/detection_project.git
cd detection_project
```

### 2. Создать виртуальное окружение (рекомендуется)

```bash
python -m venv venv
venv\Scripts\activate   # для Windows
source venv/bin/activate # для Mac/Linux
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить приложение

```bash
python app.py
```

### 5. Открыть в браузере

Перейдите по адресу:

```
http://127.0.0.1:5000
```

---

## 📸 Как использовать

1. Нажмите кнопку **«Выберите файл»** и выберите изображение (форматы JPG, PNG)
2. Нажмите кнопку **«Запустить обработку»**
3. Дождитесь результата — ноутбуки будут выделены зелёными рамками
4. Посмотрите статистику — количество обнаруженных ноутбуков
5. История всех запросов сохраняется в таблице внизу страницы
